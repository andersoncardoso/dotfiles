Some dotfiles I use:

- Neovim - neovim configuration and plugins
- Bash - bashrc modifications and aliases
- Zsh - configuration for the zshell
- Git - my gitconfig file


## Setup:

Usage: `./setup.rb [options]`

The options are:
  - `--git`
  - `--bash`
  - `--zsh`
  - `--neovim`
  - `--all` (runs all the above)
